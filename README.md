**Watchlist Manager**

Aplikasi Pengelola Daftar Tontonan Film & Serial


**Deskripsi Proyek**

Watchlist Manager adalah aplikasi desktop berbasis Java yang dirancang untuk membantu pengguna mengelola daftar film dan serial yang ingin ditonton atau sudah selesai ditonton.

Aplikasi ini menggabungkan kekuatan Spring Boot untuk manajemen backend dan database, dengan Java Swing untuk antarmuka pengguna (GUI) yang responsif. Proyek ini dibuat untuk memenuhi tugas praktikum Pemrograman Berorientasi Objek Lanjut.


**Fitur Utama**
Aplikasi ini mencakup operasi CRUD (Create, Read, Update, Delete) lengkap:
* Tambah Tontonan (Create): Menambahkan film/serial baru ke dalam daftar dengan status awal "Watchlist".
* Lihat Daftar (Read): Menampilkan semua data dalam format tabel yang rapi.
* Update Status (Update): Menandai tontonan sebagai "Completed" dan memberikan Rating Pribadi (0-10).
* Hapus Data (Delete): Menghapus data tontonan dari database secara permanen.
* Pencarian: Mencari film berdasarkan judul.
* GUI Interaktif: Menggunakan Java Swing dengan validasi input.


**Teknologi yang Digunakan**
* Bahasa Pemrograman: Java (JDK 17)
* Framework: Spring Boot 3.2.0
* Database: MySQL
* ORM: Hibernate (Spring Data JPA)
* GUI: Java Swing (JFrame)
* Build Tool: Maven
* IDE: Apache NetBeans 21 / IntelliJ IDEA
